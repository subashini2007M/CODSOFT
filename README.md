# CODSOFT Artificial Intelligence Internship

## About

This repository contains the tasks completed as part of the CODSOFT Artificial Intelligence Internship.

* Python

Description:

Technologies Used:

* Python
* Minimax Algorithm

## Task 4: Recommendation System

Description:
A movie recommendation system that suggests movies based on the user's preferred genre.

Technologies Used:

* Python
* Pandas
